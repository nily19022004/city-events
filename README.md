# Информационная система планирования городских мероприятий

Система для публикации и просмотра городских мероприятий.

## Функции:
- Просмотр афиши мероприятий
- Детальная информация о каждом мероприятии
- Административная панель для управления

## Установка:
1. Установите Python 3.10+
2. Клонируйте проект
3. Создайте виртуальное окружение: `python -m venv venv`
4. Активируйте: `.\venv\Scripts\Activate.ps1`
5. Установите зависимости: `pip install -r requirements.txt`
6. Выполните миграции: `python manage.py migrate`
7. Создайте суперпользователя: `python manage.py createsuperuser`
8. Запустите сервер: `python manage.py runserver`
